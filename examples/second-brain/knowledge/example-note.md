# Example Knowledge Note

Notes here are for things you want to remember across projects —
a library quirk, a design pattern, a lesson learned. Delete this file
once you've got the idea.

For example: "Postgres `ON CONFLICT DO UPDATE` upserts silently skip
triggers unless you name the constraint explicitly" — the kind of
thing worth keeping around, that isn't tied to any one repo.
