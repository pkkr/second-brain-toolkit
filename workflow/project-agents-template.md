# Project AGENTS.md Template

Copy this into `<project-repo>/AGENTS.md` (replace `<name>`):

    # Agent Instructions

    ## Second Brain

    Context for this project lives in the Second Brain under `~/second-brain/`:

    - Core facts & open tasks: `projects/<name>/project.md`
    - Recurring workflows: `projects/<name>/processes.md` — follow these!
    - Progress log: `projects/<name>/log.md`

    Read `~/second-brain/AGENTS.md` at the start and follow the rules
    there (maintain project notes, write the session log). If the
    Second Brain doesn't exist on this machine, just work normally.
